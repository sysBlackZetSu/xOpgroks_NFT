# Hardhat
Try running some of the following tasks:

```shell

npx hardhat compile
npx hardhat node
npx hardhat run scripts/deploy.js
or
npx hardhat run scripts/deploy.js --network bsctest
npx hardhat verify --network <address deployed>
```
